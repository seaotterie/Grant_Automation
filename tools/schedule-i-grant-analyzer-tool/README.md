# Schedule I Grant Analyzer Tool

Foundation grant-making pattern analysis from 990-PF Schedule I data.

## Features

- Grant pattern analysis (volume, size, distribution)
- Category and geographic distribution
- Recipient profiling
- Foundation intelligence and granting style
- Organization match analysis
- Strategic cultivation recommendations
