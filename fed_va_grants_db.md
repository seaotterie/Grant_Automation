## üìä API Implementation Details

### High Priority Federal APIs (Immediate Integration)

#### 1. Grants.gov API
```
Base URL: https://www.grants.gov/grantsws/rest
Key Endpoints:
  - /opportunities: Search opportunities
  - /opportunity/{id}: Opportunity details
  - /applicants: Applicant information
Authentication: API Key (free registration)
Response Format: XML/JSON
```

#### 2. USASpending.gov API
```
Base URL: https://api.usaspending.gov/api/v2/
Key Endpoints:
  - /search/spending_by_award/: Award search
  - /recipient/{duns_or_ein}/: Recipient profiles
  - /references/agency/: Agency information
Authentication: None required
Response Format: JSON
```

#### 3. SAM.gov API
```
Base URL: https://api.sam.gov/
Key Endpoints:
  - /assistance-listings/v2/: CFDA programs
  - /entity-information/v1/: Entity details
  - /opportunities/v1/: Contract opportunities
Authentication: API Key (free)
Response Format: JSON
```

#### 4. NIH Reporter API
```
Base URL: https://api.reporter.nih.gov/v2/
Key Endpoints:
  - /projects/search: Project search
  - /projects/{projectId}: Project details
  - /publications/search: Related publications
Authentication: None required
Response Format: JSON
```

### Medium Priority APIs (Phase 2)

#### FEMA OpenFEMA API
```
Base URL: https://www.fema.gov/api/open/v1/
Key Endpoints:
  - /FemaWebDeclarationsSummaries: Disaster declarations
  - /PublicAssistanceApplicants: Grant recipients
  - /HazardMitigationAssistanceProjects: Mitigation grants
Authentication: None required

```

## üîê Compliance & Legal Notes

### Public Domain Sources
- All federal government APIs (Grants.gov, USASpending, etc.) are public domain
- No attribution required, but recommended for transparency
- Rate limiting compliance required

### Terms of Service Considerations
- **GrantWatch**: Prohibits automated scraping
- **Foundation Directory**: Requires paid subscription for bulk access
- **State Agencies**: Review individual ToS, most allow reasonable use

### Data Privacy
- Grant opportunities are public information
- Organization profiles should be handled securely
- Implement appropriate caching and retention policies
- Consider GDPR implications for any EU-related data
